AKG PPL Quiz - GitHub Pages package

Upload these files/folders to the root of your GitHub Pages repository:

index.html
050_Meteorologi_fragor_och_svar.txt
060_Navigation_fragor_och_svar.txt
090_Kommunikation_fragor_och_svar.txt
050_Meteorologi_bildfragor.txt

images_quizsafe/

Image naming:
- The app automatically tries to match images by question number.
- Recommended names: images_quizsafe/050_IMG_01.png, images_quizsafe/060_IMG_08.png, etc.
- You can replace a question image by replacing the file with the same name.

Edit mode:
- Use Edit mode at the bottom next to file upload.
- You can edit questions/answers, add/delete questions, add/remove images, save locally, and export updated .txt files.
